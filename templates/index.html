About EduForecast

  • EduForecast is a web app which has a Machine Learning model running at the back.
  • The purpose of developing this app is to predict student performance in mathematics based on multiple factors.
  • The dataset is downloaded from the Kaggle.
  • Provides insights into the influence of gender, ethnicity, parental level of education, lunch type, and test preparation course on student performance.
  • User-friendly interface for inputting student information and obtaining predictions.
  • It uses various ML algorithms like KNN , Logistic Regression, SVM, Random Forest , Navie Bayes.
  • The primary goal of this project is to demonstrate the end-to-end process of developing a machine learning model and provide insights into the factors influencing student performance.
  • The codes for this project can be checked in my github repo.
